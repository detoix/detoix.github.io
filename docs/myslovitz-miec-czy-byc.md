Już teraz wiem `a C`  
Wszystko trwa, dopóki sam, tego chcesz `G D`  
Wszystko trwa, sam dobrze wiesz `a C G D`  
Że upadamy wtedy, gdy nasze życie `a C G`  
przestaje być codziennym zdumieniem... `D a C e`
